## 🌌 About The Project

From a cold, creative-less city, a small idea emerges, growing louder and louder. **The Spark** is an interactive, story-driven web experience built to demonstrate the boundaries of modern web design, combining 3D rendering, fluid animations, and a compelling narrative flow. 

Every element has been handcrafted and optimized for peak performance, ensuring a buttery-smooth 60fps experience across devices.

## 🚀 Features

- **Interactive 3D Environments:** Custom-built 3D scenes that react to scroll and user interactions.
- **Fluid Typography Animations:** Powered by GSAP SplitText & ScrambleText plugins.
- **Immersive Narrative Design:** A continuous scroll journey that tells a visual story.
- **Fully Responsive & Optimized:** Tailored layouts for desktop, tablet, and mobile viewing.

- **HTML5 & CSS3** for structural layout and core styling.

- **GSAP (GreenSock)** for high-performance animations and scroll-trigger effects.
- **WebGL / 3D Integrations** to build the immersive visual world.
